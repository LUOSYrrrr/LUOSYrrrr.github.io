1.2指令要求解读

‘Summarize the information by selecting and reporting the main features, and make comparisons where relevant’

指令关键词

图表的 Features 是什么？

我们需要对 Features 做的动作

如何对图标中的 Features 做 Select、Report 及 Make comparison 的动作？

拒绝地毯式平铺，所有特征不能全写，要找main features

找到main features 的方法就是 作比较

1.2.1作比较的三种常见维度

（Make comparison）

a.数值方面

b.趋势方面

对趋势的提取只适用于动态图

上升，下降，波动的区别

上图不是波动，波动：数据围绕一个固定值上下变化

c.幅度方面

比较单独两个数值，用大或小形容幅度

描述趋势的变化，用剧烈或平缓形容幅度

1.2.2main features总结

动态图都可以写，静态图没有趋势

通常四段

第一段：改写段

第二段：概述段

第三段和第四段：细节段

图表 main features，也可以只有一个细节段

也可以把overview放最后
