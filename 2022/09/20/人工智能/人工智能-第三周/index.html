1.选择一个分类算法

“没有免费的午餐”NFL理论

每一种算法都有其特定的前提，没有适用于所有场景的算法

通过性能比较选择最适合的算法模型

NFL定理的重要意义在于，不能在脱离实际意义情况下，空泛地谈论哪种算法好，这是毫无意义的，没有适用于所有场景的算法，要谈论算法优劣必须针对具体的学习问题。

2.逻辑斯蒂回归

其实是分类模型、非回归模型；解决的是：线性二元分类问题；业界广泛采用

逼近的是类标的函数

函数：

几率

几率的意义 —— 正例发生与不发生的相对可能性

p：正率发生的概率

1-p：正率不发生的概率

对数几率函数：对“几率”取自然函数，输入0~1，具有S形状

正例出现的相对可能性的对数
对比：“类别”和“类别出现的概率”

逻辑斯蒂回归:预测输出逼近真实类标的对数几率,，即 ln(p/1-p)=z →p=1/(1+e-z)，是正例的概率，即判断“输入z=wTx”属于正例的概率。

用“对数函数”联系了“预测输出”和“真实类标的几率”

逻辑斯蒂函数：Sigmoid函数，S型函数

激活函数：

激活输出：判别“输入”属于正例的概率\

激活输出值在0、1之间

线性分类对比sigmoid

二元输出：0、1

Cost function: 最小化代价函数，使用梯度下降GD/SGD ……

1、likelihood function L(w) 似然函数L(w)，某种事件发生的可能性，从观测结果到有关事件的参数估计，表达的本质：由估计参数决定的样本属于其真实类标的概率
2、log-likelihood function 对数似然函数l(w)：积  和，最大化对数似然的本质：每个样本属于其真实类标的概率越大越好
3、最小化代价函数J(w)：注意采用负 “-”之后，最大化对数似然，转化为“最小化代价函数”，以便使用梯度下降法

正则化解决过拟合

用 λ 限定W值 —— 学到的知识

W值随C变化的L2正则化路径图

C 是 λ 的倒数
C 越小正则化强度越大，
防止过拟合

4.决策树

分而治之

实践中的方案：从根节点开始做样本划分

the largest Information Gain (IG) 最大化信息增益

—— 节点上的样例尽量属于同一类
—— 直至同一类 ？？？
—— 过拟合 ？？？

剪枝以限定树的最大深度

最大化信息增益 —— 目标函数:

父、子节点的不纯度之差

f : the feature to perform the split 用于划分数据集的特征
Dp and Dj : the dataset of the parent and jth child node 数据集
I : node impurity measure 节点不纯度度量，有三种基本类型
Np : the total number of samples at the parent node 样本数
Nj : the number of samples in the jth child node 样本数

ID3决策树：最大化信息增益，有偏好于属性取值多的
C4.5决策树：选择划分属性，先最大化信息增益（偏好属性取值多的），再结合“信息增益率”（偏好属性取值少的），“信息增益率”是信息增益与父节点经验熵之比

二叉决策树

p(i|t) is the proportion of the samples that belong to class i for a particular node t. 节点t上第 i 类样本所占比例。

信息熵：本质含义是最大化树中的互信息，节点纯度越高值越小。最小/最大值：节点所有样例同属于一类时取值最小，为0；节点所有样例均衡分布时取值最大，为1。

基尼系数：本质含义是最小化错分的概率，节点纯度越高值越小。选择划分后基尼系数最小的属性进行划分。最小/最大值：节点所有样例同属于一类时取值最小，为0；节点所有样例均衡分布时取值最大，对二分类为0.5。

分类误差：max{p(i/t)}代表节点的主导类，剪枝时使用。最小/最大值：节点所有样例同属于一类时取值最小，为0；节点所有样例均衡分布时取值最大，对二分类为0.5。
