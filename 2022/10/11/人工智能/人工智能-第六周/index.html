利用特征相关性降维，提高存储、计算效率，改善预测能力，特别是在无法使用正则化时解决数据的维数灾难问题。

这章主要讲特征抽取（提取）

1. PCA 主成分分析

Principal Component Analysis

不是直接移除列，是找最关键的，然后投影到另一个空间上，做分析，做分类

1、基于特征相关性压缩数据，去除数据冗余，实现降维；
2、数据最大方差方向，特征最不相关。
3、进一步新增的数据方差方向，要求与原有的方向线性无关 —— 正交，保证独立性，不能由原有的线性组合得出。

Unsupervised linear transformation technique 无监督线性转换

Based on the correlation between features 基于特征相关性

The directions of maximum variance in high-dimensional data
寻找高维数据中的最大方差方向—— 特征最不相关

x1 and x2: original feature axes

PC1 and PC2: principal components—— PC1、PC2 要求线性无关，正交（自身变异最大的方向）

z=xW, z<=1

1、W：dxk ‘s eigenvector projection matrix（特征向量投影矩阵）
2、The orthogonal component axes of maximum variance in a dataset （数据集中最大方差的正交分量轴 ）

注意 k<=d

怎么去找W？

PCA基本实现方法

将d维转换成k维

1.第一步，标准化

2.第二步，构造协方差矩阵

1、对角线上分别是每个特征 xi 的方差，非对角线上是 两个特征xj 和 xk 的协方差，d维的方阵。代表样本的“多样性”。
2、方差用来度量单个变量“自身变异”的大小，方差越大变量的变异越大
3、协方差用来度量两个变量之间“协同变异”的大小，即两个变量相互影响的大小，协方差的绝对值越大，两个变量相互影响越大，反之越小。
1）大于0表示若有一个增，另一个也增，变化一致；
2）小于0表示若有一个增，另一个则减，变化相反；
3）为0表示两者独立，相互间无影响。

3.第三步，对协方差矩阵进行特征分解，获取特征向量、特征值

构造关于协方差矩阵的特征方程

https://blog.csdn.net/yangleo1987/article/details/52845912
1、一个矩阵的特征方程：求能被该矩阵变换并拉缩 λ 倍的向量 v，该向量称为特征向量，对应的 λ 值称为特征值
2、λ is a scalar: the eigenvalue. 特征值是标量
3、同一个特征值，对应一类特征向量
4、各不相等的特征值，对应的特征向量线性无关。成为一组变换“基”
5、不相等的两个特征值对应的特征向量正交

4.第四步，特征向量依据特征值排序

5.第五步，选择前k个特征值对应的特征向量

6.第六步，由“前”k个特征向量构造投影矩阵w

7.第七步，进行可视化

左边：原数据分布

右边：变换后数据分布

左边：变化后每一个特征维的可解释方差、累积可解释方差

右边：变换后数据分布

解释方差率：数据的表征性

2. LDA 线性判别分析

Linear Discriminant Analysis

LDA一般指隐含狄利克雷分布。 隐含狄利克雷分布（Latent Dirichlet Allocation，LDA），是一种主题模型（Topic Model），它可以将文档集中每篇文档的主题按照概率分布的形式给出。我们将在第八章情感分析里讲解和使用主题模型。
本章的LDA是线性判别分析，用于数据降维。

（FDA）也是这个，基本一致

Supervised linear transformation technique 有监督线性转换

The data is normally distributed 假设数据是正态分布的

The classes have identical covariance matrices 类具有相同的 Σ

Optimizes class separability 优化类的可分性（类间/类内）

1、Find the feature subspace that optimizes class separability （寻找可以优化类可分性的特征子空间 ）
2、As far as possible between classes, as close as possible within the class. 类间尽可能远，类内尽可能近。

两个类最大方差方向相同，且数据是正态分布的，若往最大方差方向上投，就不能分开了

FDA情况

用lda就能分开了，投影到黄色线能分开

LDA基本实现方法

1.第一步：标准化

2.第二步：对每一个类，找平均向量

ni，为样本数

3.第三步：计算类间/类内散步矩阵（与数据分布相关的矩阵

类间为Sb，类内为Sw，mi为第i类平均向量

si为d*d

4.第四步:Sw要尽量小，Sb尽量大

5.第五步：选择前k个特征值对应的特征向量

6.第六步：由“前”k个特征向量构造投影矩阵W

7.第七步：利用W投影数据集到新特征空间

前两个线性判别式就捕获了葡萄酒训练数据集中100%的有用信息

3. KPCA 核主成分分析

Kernel Principal Component Analysis

针对线性不可分，非线性可分

变化之后才能找到线性超平面

Kernelized version of PCA 核化版的PCA
Nonlinear separable problems 针对非线性可分问题
Transform datasets onto a new, lower-dimensional linear separable subspace 映射到一个低维的线性可分子空间

在核PCA之后样本已经投影到各个分量上，而不是像标准PCA方法那样需要构造一个投影矩阵W对样本进行投影变换。

特征向量不是主分量轴，而是已经投射到这些轴上的样本。

KCPA基本实现方法

γ=1/2σ2

K为n*n

K‘为n*n

取前k个，为n*k的

不用转换矩阵了
